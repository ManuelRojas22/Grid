# 🧱 CSS GRID EN LA ESTRUCTURA DEL PROYECTO

## 📄 Descripción

En este proyecto se utilizó **CSS Grid** como sistema principal de maquetación para organizar la estructura de la página. Esta tecnología permite dividir la interfaz en una cuadrícula formada por filas y columnas, lo que facilita posicionar cada sección del sitio de forma clara y ordenada.

El uso de Grid permite estructurar la página de manera más limpia y mantener una distribución visual organizada.

---

## 🎯 ¿Por qué se utilizó CSS Grid?

CSS Grid fue implementado para definir la **estructura principal de la interfaz** y permitir una distribución controlada de los elementos dentro del diseño.

Las razones principales para utilizar Grid fueron:

- Organizar la página en una estructura clara.
- Separar las diferentes secciones del sitio.
- Distribuir los elementos utilizando filas y columnas.
- Mantener una presentación visual ordenada.

---

## 🧩 Organización del layout

Dentro del Grid se organizan las principales áreas del sitio:

- **Menú de navegación**
- **Encabezado**
- **Contenido principal**
- **Secciones de productos**

Cada sección ocupa una posición específica dentro de la cuadrícula definida en el diseño.

---

## ⚙️ Beneficios del uso de Grid

El uso de CSS Grid dentro del proyecto permite:

- Crear una **estructura visual organizada**.
- Mantener el código más **claro y fácil de mantener**.
- Controlar la posición de cada elemento dentro del layout.
- Facilitar modificaciones en la estructura del diseño.

---

## 🛠️ Tecnologías utilizadas

- **HTML5** – estructura del contenido  
- **CSS3** – estilos y maquetación mediante Grid y Flexbox  
- **JavaScript** – interacciones básicas
